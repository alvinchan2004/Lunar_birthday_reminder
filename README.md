# Market_Pulse
This project automatically generates daily reports summarizing the most recent period in the stock market. It uses multiple analysis instruments and sector news to identify stocks with strong profit potential.

The project includes the following components:
1. A scheduler that runs on a server (Docker or VM)
1. Stock data ingestion
1. Market-trend overview generation
1. Indicator-based stock selection
1. News-based stock selection
1. Report generation



## How to use
Use setup.bat to build venv.<br>
Use VS Code to develop the code.
